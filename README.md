# Dovahzul translator

## Summary

A translator app for English <-> Dovahzul built in svelte and based on the work by [Thuum.org](https://www.thuum.org/).

## Quickstart

1. `npm i`
2. `npm run dev`

## Deployment

Check out the deployed app online at https://dovahzul-translator-jg.netlify.app/
